This repository contains a theme for the Touying presentation package for Typst. It is designed to produce both slides and a handout from the same content data file, replicating a workflow that used LaTeX Beamer and Beamer-Article.  I use it primarily to create materials for lectures. This repository contains the theme and a sample presentation that I hope explains the basic capabilities of the theme.

I had no idea how to start, so I had to get help from Claude. The code is fairly easy to follow, however.
